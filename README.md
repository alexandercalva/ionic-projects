Instrucciones para ejecutar la aplicación requerida
•	Habilitar el celular en modo desarrollador
•	Conectar con un cable USB al computador
•	Abrir Android Studio
•	Abrir la carpeta llamada Android (path: busmart-project/platforms)
•	Ejecutar el simulador del programa (se puede descargar la aplicación al celular directamente)
